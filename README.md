# node-mongo-rest-api

production ready server code

### Initial setup

1. Create .env file in root directory with following env variables

```
    PORT=
    CLUSTER_MODE= bool
    TZ=UTC or any timezone

    MONGODB_URL=
    DB_NAME=
    DB_USER=
    DB_PASS=

    ACCESS_TOKEN_SECRET=
    REFRESH_TOKEN_SECRET=

    SALT=
```
